# 📊 Amazon Products Dashboard

A Power BI dashboard analyzing Amazon product data 
across multiple categories.

## 🔍 Business Questions Answered
- Which categories have the highest average prices?
- What is the relationship between rating and discount?
- Which products are top-rated by customers?

## 💡 Key Insights
- Electronics leads in average actual price
- Higher discounts don't always correlate with better ratings
- Products rated 3.0 show the highest discount percentages

## 🛠️ Tools Used
- Power BI
- CSV Dataset (Amazon Products)

## 📁 Files
- `Amazon Dashboard.pbix` - Power BI file
- `amazon.csv` - Raw dataset
